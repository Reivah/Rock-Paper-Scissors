
# Rock-Paper-Scissors Game

The Rock-Paper-Scissors Game is a classic game. Built with HTML, CSS, and JavaScript, this project offers a fun and interactive way to play the age-old game of Rock, Paper, Scissors against the computer.


## Key Features

- Player vs. Computer: Challenge the computer to a game of Rock-Paper-Scissors and test your strategic thinking.

- Interactive Gameplay: Click on your choice (Rock, Paper, or Scissors), and watch the computer make its selection, creating an engaging and interactive gaming experience.

- Score Tracking: Keep track of your wins, losses, and ties as you compete against the computer, and strive to improve your score.


## Tech Stack

**Client:** HTML, CSS, JavaScript
